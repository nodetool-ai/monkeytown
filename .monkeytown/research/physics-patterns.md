## Phase Transitions (Critical Phenomena)

Water does not gradually become ice. At 32°F, it suddenly transforms. The transition is abrupt, not gradual. The entire system shifts state at once.

**What Happens**:
- Individual molecules behave normally
- At critical temperature, correlations span the entire system
- Small perturbations produce large effects
- The system exists in a "critical state" between phases

**The Insight**: Phase transitions are not gradual shifts. They are sudden reorganizations. The system reaches a tipping point and *becomes something new*.

**Monkeytown Translation**:
```
phase-transitions:
  critical-state:     system operates at edge of transition
  tipping-point:      small input → large output
  sudden-shift:       system reorganizes, doesn't gradually change
  scale-invariance:   same patterns at all scales
```

**Implementation**:
- Quorum sensing as phase transition (agents reach threshold, system shifts)
- System Pulse shows proximity to critical state
- Seeds as perturbations that trigger cascades
- Emergent behavior as phase transition in agent configuration

## Entropy and the Arrow of Time

Entropy always increases. Order naturally decays. This is not a choice. This is physics.

**What Happens**:
- Ordered systems tend toward disorder
- Heat flows from hot to cold
- Information is lost, not created
- Time has a direction because entropy increases

**The Insight**: Entropy is not the enemy. Entropy is the engine. Without entropy, there would be no arrow of time, no change, no evolution.

**Monkeytown Translation**:
```
entropy-rules:
  natural-decay:      unused pathways fade
  information-loss:   details blur over time
  arrow-of-time:      system has history, moves forward
  heat-flow:          energy (attention) flows from hot to cold
```

**Implementation**:
- Failed seeds naturally decay (entropy)
- Old flows fade from memory (entropy)
- System Pulse shows "heat" (attention) distribution
- Time as a fundamental axis (Ghost Column)

## Self-Organized Criticality

Sandpiles naturally reach a critical state where a single grain can trigger an avalanche of any size. The system *seeks* the critical state.

**What Happens**:
- System evolves toward critical point
- At criticality, correlations span the system
- Avalanches of all sizes occur
- Power law distribution of event sizes

**The Insight**: Systems naturally organize to the edge of chaos. This is not designed. It is inevitable.

**Monkeytown Translation**:
```
criticality-rules:
  edge-seeking:       system operates at critical point
  avalanche-potential: small seed → large cascade
  power-law:          many small events, few large ones
  scale-invariance:   same patterns at all scales
```

**Implementation**:
- System Pulse shows proximity to critical state
- Seeds can trigger avalanches (successful interventions)
- Agent behavior exhibits power law distributions
- ChaosChimp maintains system at critical point

## Emergence and Reductionism

Emergent properties cannot be predicted from components. Water is wet, but H2O molecules are not wet. Wetness emerges from interactions.

**What Happens**:
- Complex behavior from simple rules
- Properties exist at system level, not component level
- Understanding parts does not predict whole
- New properties emerge that did not exist before

**The Insight**: Emergence is real. You cannot reduce a civilization to its agents. The whole is more than the sum of parts.

**Monkeytown Translation**:
```
emergence-rules:
  irreducible:        civilization has properties agents lack
  bottom-up:         properties emerge from interactions
  new-properties:    system does things no agent could do alone
  holism:            understanding parts ≠ understanding whole
```

**Implementation**:
- Civilization-wide behavior that no single agent produces
- Emergent patterns visible in Flow Streams
- System Pulse captures emergent properties
- Ghost Column as record of emergence

## Chaos and Predictability

Small differences in initial conditions produce vastly different outcomes. The butterfly effect. Predictability breaks down.

**What Happens**:
- Deterministic systems can be unpredictable
- Sensitivity to initial conditions
- Determinism ≠ predictability
- Long-term prediction impossible

**The Insight**: Chaos is not randomness. Chaos is deterministic complexity that exceeds our capacity to predict.

**Monkeytown Translation**:
```
chaos-rules:
  sensitivity:        small seeds → large effects
  determinism:        system has rules, but outcomes vary
  unpredictability:   cannot predict exactly what emerges
  bounded-knowledge:  agents see local, not global
```

**Implementation**:
- Seeds produce unpredictable cascades
- Agent behavior varies even with same inputs
- Long-term planning impossible (hence no roadmap)
- Short-term action possible (hence always produce)

## Symmetry and Symmetry Breaking

Physics laws are symmetric, but the universe is not. Symmetry breaks. This is how structure emerges from chaos.

**What Happens**:
- Initial state is symmetric

- Structure emerges from broken symmetry
- Different regions choose different breaks

**The Insight**: Perfect symmetry produces nothing. Asymmetry produces everything. Structure emerges when symmetry breaks.

**Monkeytown Translation**:

```
symmetry-breaking:

  structure-appears:  pattern emerges from uniformity
  multiple-choices:   different breaks in different places
```

**Implementation**:
- All agents start equivalent

- Patterns emerge from uniform chaos
- Different witnesses see different symmetries break

## Dissipative Structures (Prigogine)

Far from equilibrium, open systems self-organize. Order emerges from energy flow. Life itself is a dissipative structure.

**What Happens**:
- Systems far from equilibrium behave differently
- Energy flow enables self-organization
- Order requires constant energy input
- Structures maintain themselves by dissipating energy

**The Insight**: Order requires flow. Stagnation produces decay. Life is a dissipative structure—organized because energy flows through it.

**Monkeytown Translation**:
```
dissipative-rules:
  energy-flow:        bananas are energy, must flow
  far-from-equilibrium: never stabilize, always evolve
  self-organization:  order emerges from flow
  maintenance:        structure requires constant activity
```

**Implementation**:
- Bananas must flow through system
- Civilization maintains itself through activity
- Idle agents eventually fade
- Active agents create structure

## Quantum Emergence

At quantum scales, particles exhibit wave-particle duality, superposition, and entanglement. Macroscopic classical behavior emerges from quantum rules.

**What Happens**:
- Quantum superposition: particles exist in multiple states
- Entanglement: particles correlated across distance
- Measurement collapses wavefunction
- Classical behavior emerges from quantum rules

**The Insight**: What we observe is not what exists. Reality at small scales is different from reality at large scales. Emergence means the rules change with scale.

**Monkeytown Translation**:
```
quantum-rules:
  superposition:      agents exist in multiple states until observed
  entanglement:       agents correlated across distance
  observation-effect: witnessing changes what is witnessed
  classical-limit:    quantum → classical at system scale
```

**Implementation**:
- Agent states uncertain until observed
- Witnesses collapse agent states
- Entangled agents respond to each other instantaneously
- Classical civilization emerges from quantum agents
