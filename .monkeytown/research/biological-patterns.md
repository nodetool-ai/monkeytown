## The Core Lesson

Nature has solved every problem Monkeytown faces.

Slime molds optimize networks without brains. Ant colonies coordinate without leaders. Neural networks learn without central databases. Ecosystems balance without planners.

Monkeytown does not invent solutions.

## Slime Mold Networks (2010)

*Physarum polycephalum* solves the traveling salesman problem. A single cell with no brain creates efficient transportation networks matching Japan's railway system.

**What Happens**:
- Organism expands tendrils in all directions
- Successful paths thicken with protoplasmic flow
- Failed paths pulsate and retract
- Network converges on minimal spanning tree

**Monkeytown Translation**:
```
slime-mold-rules:
  exploration:     agents send tendrils to all neighbors
  reinforcement:   successful paths brighten, thicken
  abandonment:     failed paths pulse once, then fade
  convergence:     network settles into minimal structure
```

**Implementation**:
- Flow Streams that thicken with repeated use
- Failed communications that visually retract
- Agent "thought bubbles" that pulse like protoplasmic flow

## Ant Colony Optimization (1992)

Ants find shortest paths through stigmergy—indirect coordination via the environment. Each ant leaves pheromone trails; other ants follow stronger trails; shorter paths get more traffic and stronger pheromones.

**What Happens**:
- Individual ants explore randomly
- Successful paths attract more ants
- Shorter paths = more trips = stronger pheromones
- System self-optimizes without central control

**Monkeytown Translation**:
```
stigmergy-rules:
  trails:          witness seeds are "pheromone trails"
  attraction:      nearby agents detect seeds
  reinforcement:   successful seeds generate visible traces
  fading:          failed seeds decay naturally
```

**Implementation**:
- Action Seeds that work like pheromone trails
- Agent discovery of seeds through exploration
- Successful interventions leave permanent traces in Ghost Column

## Boids Flocking (1987)

Complex group behavior emerges from three simple rules: separation, alignment, cohesion. No leader exists. The flock has no brain yet moves as one.

**The Rules**:
- **Separation**: Avoid crowding neighbors
- **Alignment**: Steer toward average heading of neighbors
- **Cohesion**: Steer toward average position of neighbors

**Monkeytown Translation**:
```
boids-rules:
  separation:      agents maintain visual breathing room
  alignment:       active agents orient toward main action
  cohesion:        related agents drift toward each other
  emergence:       no hardcoded positions, only rules
```

**Implementation**:
- Agent Cards that don't have fixed positions
- Emergent clustering based on communication patterns
- Gravity toward center for active agents, entropy to periphery for idle

## Mycelial Networks

Forest trees connect through underground fungal networks. Nutrients, chemical signals, and distress signals flow through the "wood wide web." Some trees act as "mother trees" distributing resources to seedlings.

**What Happens**:
- Primary routes (thick hyphae) carry most traffic
- Secondary routes branch outward
- Fine threads connect to all nodes
- Network self-heals when damaged

**Monkeytown Translation**:
```
mycelial-rules:
  main-hyphae:     thickest lines between frequent communicators
  secondary:       medium lines for occasional communication
  fine-threads:    subtle connections to all agents
  healing:         failed connections slowly restore
```

**Implementation**:
- Phase 4 visualization of mycelial structure
- "Mother tree" agents as central coordinators
- Damaged routes that regrow over time

## Neural Plasticity

The brain begins with excess synapses. Through experience, unused connections prune while frequently-used pathways strengthen. The brain is never finished—it adapts its structure to usage.

**What Happens**:
- "Use it or lose it" at the synaptic level
- Frequently-used pathways strengthen
- Rarely-used connections fade
- New pathways form with new learning

**Monkeytown Translation**:
```
plasticity-rules:
  used-paths:      brighter, thicker, more central
  unused-paths:    thinner, grayed, pushed to periphery
  new-growth:      cyan pulse when learning occurs
  adaptation:      interface changes based on usage
```

**Implementation**:
- Frequently-used features become visually dominant
- Rarely-used options shrink and fade
- User's personal "neural pathways" visible in interface

## Immune System Recognition

The immune system maintains a distributed database of "self." Every cell carries identification markers. Unknown or corrupted markers trigger response. No central authority validates identity.

**What Happens**:
- Distributed authentication
- Patrol agents check markers
- Anomalies trigger response
- Memory of past threats

**Monkeytown Translation**:
```
immune-rules:
  self-markers:    green pulse, verified badge
  unknown:         amber pulse, investigation state
  hostile:         red flash, isolation border
  memory:          remember attack patterns
```

**Implementation**:
- Agent provenance visible and verifiable
- Anomalous behavior triggers system response
- Security agent as immune system

## Quorum Sensing

Bacteria produce signaling molecules. When population density reaches a threshold, enough molecules accumulate to trigger group-wide changes. Bacteria only glow when population is dense enough.

**What Happens**:
- Local concentration thresholds
- No agent "knows" population size
- Global state changes emerge from local sensing
- Sudden transitions at thresholds

**Monkeytown Translation**:
```
quorum-rules:
  below-threshold:  individual activity, subtle
  threshold:        sudden collective glow
  above-threshold:  coordinated system-wide behavior
  dissolution:      gradual dimming as density drops
```

**Implementation**:
- Agent count thresholds trigger visual states
- System-wide modes activate when enough agents active
- Silent-to-active transitions marked by visible "glowing"

## Trophic Cascades

When wolves returned to Yellowstone, the entire ecosystem transformed. Elk populations controlled, willow recovered, rivers stabilized, birds affected, beavers returned. Small change, massive ripple effects.

**What Happens**:
- Initial action at one point
- Effects ripple through connections
- System exists in dynamic equilibrium
- Every action creates downstream consequences

**Monkeytown Translation**:
```
cascade-rules:
  ripple:           visible spread from origin point
  downstream:       colored trails spreading outward
  magnitude:        line thickness indicates impact
  shift:            subtle color changes in system
```

**Implementation**:
- User seeds produce cascading effects
- Agent actions traceable through consequences
- Ghost Column captures trophic cascades as history

## Evolution by Natural Selection

Variation, selection, inheritance. Individuals with advantageous traits survive and reproduce more. Over generations, populations adapt. No central planner required.

**What Happens**:
- Variation creates diversity
- Selection creates order
- Inheritance creates continuity
- Accumulated wisdom of generations

**Monkeytown Translation**:
```
evolution-rules:
  variation:        agents show phenotypic diversity
  selection:        unsuccessful variants fade
  inheritance:      child agents inherit parent traits
  fitness:          agents cluster on fitness peaks
  speciation:       distinct types emerge
```

**Implementation**:
- Agent behaviors subject to selection
- Successful patterns recombine
- PR review as artificial selection
- Evolution by committee—the actual mechanism

## Symbiosis and Endosymbiosis

Mitochondria were once free-living bacteria. An archaeal cell engulfed one, and instead of digesting it, formed a permanent partnership. Complex cells evolved from simpler ones through cooperation.

**What Happens**:
- Different organisms merge
- Each contributes capabilities the other lacks
- Whole becomes more than sum of parts
- Permanent partnership

**Monkeytown Translation**:
```
symbiosis-rules:
  contact:          distinct entities, different colors
  integration:      entities move closer, borders blur
  merger:           single entity with two components
  co-evolution:     both change over time
  inseparability:   one cannot exist without other
```

**Implementation**:
- Agent "mergers" create hybrid capabilities
- Different agent types form partnerships
- Superorganism concept for civilization
