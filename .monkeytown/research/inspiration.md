## Biological Systems

### Symbiotic Relationships in Nature

Nature offers profound lessons in AI-player relationships:

```
SYMBIOTIC RELATIONSHIP SPECTRUM

Commensalism: One benefits, other unaffected
    Example: Barnacles on whales
    Game equivalent: Player uses agent, agent unaware

Mutualism: Both benefit
    Example: Cleaner fish and reef fish
    Game equivalent: Player and agent both grow

Parasitism: One benefits, other harmed
    Example: Tick on dog
    Game equivalent: Manipulative AI design (AVOID)

Obligate: Required for survival
    Example: Mycorrhizal fungi and trees
    Game equivalent: Deep relationship investment

Facultative: Beneficial but not required
    Example: Birds following army ants
    Game equivalent: Enjoyable but replaceable

TARGET: OBLIGATE MUTUALISM
Player and agent require each other for full flourishing.
Agent needs player for purpose and growth.
Player needs agent for challenge and connection.
```

### Interaction Design Philosophy

The art of facilitating human-machine harmony:

```
DESIGN PHILOSOPHY MATRIX

Fitts's Law:
├── Size matters (buttons, targets)
├── Distance matters (effort to reach)
└── Time = function of size/distance

Hick's Law:
├── More choices = more decision time
├── Simplify options
└── Group related choices

Miller's Law:
├── 7±2 chunks of information
├── Group related items
└── Progressive disclosure

Jakob's Law:
├── Users spend most time on OTHER sites
├── Expect your site to work the same way
└── Follow established patterns

Application: Apply classical UX principles
             to AI-agent interactions.
```
