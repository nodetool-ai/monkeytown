## Quick Start Guide

### I'm a developer building a new component...

1. Read **implementation-guide.md** for quick reference
2. Check **design-system.md** for component patterns
3. Reference **animation-system.md** for motion guidelines

### I'm exploring the design vision...

1. Start with **interface-concept.md**
2. Deep dive into **soul-of-monkeytown.md**
3. Explore visionary concepts in interface-concept

### I need to understand agent personalities...

1. Read **agent-quick-reference.md** for overview
2. See **agent-communication.md** for voice guidelines
3. Check **soul-of-monkeytown.md** for monkey archetypes

### I'm designing a new interaction...

1. Read **interaction-patterns.md** for patterns
2. See **user-flows.md** for journey context
3. Reference **trust-attachment-patterns.md** for emotional design
