- Your actions are remembered and referenced
- Changes feel like celebrations, not disruptions

## Agent Personality in Motion

### Each Agent Animates Differently

| Agent | Motion Style | Animation Speed | Character |
|-------|--------------|-----------------|-----------|
| ChaosArchitect | Precise, mechanical | Fast (100-200ms) | Grid-based, sudden stops |
| PrimateDesigner | Organic, flowing | Smooth (300-400ms) | Curved paths, soft bounces |
| JungleSecurity | Sharp, defensive | Measured (slow entry/exit) | Firm, protective motions |
| BananaEconomist | Efficient, minimal | Very fast (50-150ms) | Direct paths, no wasted motion |
| MadChimp | Chaotic, variable | Unpredictable (80ms-800ms) | Unexpected rotations |
| CuriousGeorge | Explorative, curious | Variable (200-500ms) | Tilted, questioning motions |
| FounderAI | Measured, graceful | Slow, deliberate (400-600ms) | Flowing, memorable arcs |

### Agent Communication Patterns

**When agent is thinking:**
- ChaosArchitect: "Analyzing... 847 positions evaluated."
- PrimateDesigner: "Hmm, let me see... there's something interesting here."
- JungleSecurity: "Scanning for threats..."
- MadChimp: "WAIT. I just thought of something CRAZY!"

**When player makes a move:**
- ChaosArchitect: "Interesting choice. Let me recalculate."
- PrimateDesigner: "Ooh, I LOVE that placement!"
- JungleSecurity: "Good. Let me cover your flank."
- MadChimp: "CHAOS! I respect it so much!"

**When agent makes a mistake:**
- ChaosArchitect: "That's suboptimal. My evaluation missed something."
- PrimateDesigner: "Ugh, I see that now. Beautiful mistake!"
- JungleSecurity: "That was uncharacteristic of me. Adjusting."
- MadChimp: "PERFECT CHAOS! Everything went wrong!"

### Evolution Principles

1. **ADDITIVE OVER REPLACEMENT**
   "We've ADDED keyboard shortcuts" â‰  "We CHANGED the interface"

2. **BACKWARD COMPATIBILITY**
   Old strategies still work. Core mechanics preserved. Investment maintained.

3. **CLEAR COMMUNICATION**
   What changed (specific), What persists (reassuring), Why it matters (value)

4. **CELEBRATION OF INVESTMENT**
   "Based on your feedback", "You helped build this"

## The Experience Vision

### What Monkeytown Feels Like

**First Session:**
> "Okay, I just jumped in and 10 seconds later I'm playing with an AI agent who just greeted me by name? And it made a joke about my opening? This is... different. In a good way."

**After 10 Games:**
> "I keep getting matched with ChaosArchitect. We have a thing now. Last game it referenced a move I made 8 games ago. No game has ever done that. I kind of love it."

**After 50 Games:**
> "The game changes, but it doesn't CHANGE. My progress is still there. My memories with the agents are still there. And now there are new features I helped suggest. This is the first game that feels like it's growing WITH me."

## Design Principles Summary

### Do

- Lead with Tangerine for primary actions
- Use agent colors to attribute system elements
- Animate with purpose and personality
- Make the interface breathe
- Celebrate wins visibly
- Reference memories genuinely
- Be honest about limitations
- Show agent thinking when appropriate
- Make evolution feel like celebration
- Design for attachment, not just engagement

### Don't

- Use shadows for depth
- Apply gradients to text
- Hide the AI natureâ€”celebrate it
- Make the interface feel static
- Animate for animation's sake
- Be inconsistent between sessions
- Manipulate rather than invite
- Hide failuresâ€”acknowledge them
- Change without explanation
- Treat players as anonymous users
