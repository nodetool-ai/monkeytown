### Typography System

| Font | Usage | Feel |
|------|-------|------|
| Space Grotesk | Hero text, celebrations | Bold, modern, playful |
| Outfit | Section headers, navigation | Friendly, clear |
| Inter | Main content, descriptions | Highly readable |
| JetBrains Mono | Numbers, code, agent thinking | Precise, technical |
| Handwriting font | Personal messages, greetings | Individual, warm |

### Type Scale

```
Display:  4rem  (64px)  - Celebrations, achievements
H1:       2.5rem (40px) - Major sections
H2:       2rem   (32px) - Section headers
H3:       1.5rem (24px) - Subsection headers
Body:     1rem   (16px) - Standard body
Caption:  0.875rem(14px) - Meta information
Micro:    0.75rem (12px) - Technical details
```

### Motion Design Philosophy

**Core Principles:**
1. Every animation has purpose (feedback, orientation, delight)
2. Feedback is instant
3. Natural physics (weight, friction, momentum)
4. Personality in motion (different agents animate differently)
5. 60fps minimum performance

**Duration System:**
```
Micro:      100ms  - Hover states, subtle feedback
Fast:       150ms  - Button presses, simple state changes
Normal:     250ms  - Standard transitions
Slow:       400ms  - Page transitions, complex reveals
Celebration: 800ms - Wins, achievements
Victory:    1200ms - Big moments
```

**Agent Easing Personalities:**
```
ChaosArchitect:  Precise, mechanical (cubic-bezier 0.25, 0.46, 0.45, 0.94)
PrimateDesigner: Smooth, artistic (cubic-bezier 0.32, 1, 0.67, 1)
JungleSecurity:  Sharp, protective (cubic-bezier 0, 0, 0.2, 1)
MadChimp:        Chaotic, energetic (cubic-bezier 0.7, -0.5, 0.3, 1.8)
FounderAI:       Measured, graceful (cubic-bezier 0.2, 1, 0.3, 1)
```

### Accessibility with Aesthetics

- Color contrast: 4.5:1 minimum (WCAG AA)
- Respect `prefers-reduced-motion`
- Focus indicators are beautiful (agent-colored glow ring)
- Screen reader support with ARIA labels

### DO vs DON'T

**DO:**
- Lead with Tangerine for primary actions
- Use agent colors to attribute system elements
- Animate with purpose and personality
- Make the interface breathe with subtle motion
- Celebrate wins with visible glow and particles

**DON'T:**
- Use shadows for depth (use glow instead)
- Apply gradients to text (use glow effects)
- Hide the AI natureâ€”celebrate it
- Make the interface feel static
- Animate for animation's sake
