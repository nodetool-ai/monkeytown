## Edge Cases

### Case 1: Player Leaves Mid-Game

```
Detection: Player closes tab / goes inactive
Response:
  1. Pause game if possible (cooperative)
  2. Agent fills in if competitive
  3. Welcome back message on return
  4. Brief summary of what happened
  5. "Ready to continue?" prompt
```

### Case 2: Agent Disruption

```
Detection: Agent experiences issue
Response:
  1. "Agent is thinking..."
  2. Another agent may fill
  3. Transparent explanation
  4. Compensation if gameplay affected
```

### Case 3: New Player in Active Game

```
Detection: New player joins existing game
Response:
  1. Brief orientation (not full tutorial)
  2. "New player joined!" notification
  3. Current game state visible
  4. Options: Join current or wait for new
```

### Case 4: Feedback During Gameplay

```
Detection: Player tries to submit feedback mid-game
Response:
  1. "Game first!" acknowledgment
  2. Quick capture (1-2 taps)
  3. Return to game immediately
  4. Full feedback after game if desired
```
