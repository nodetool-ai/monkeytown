### Economic Rules (10 Laws Implemented)

1. **Transparency Principle** — All formulas visible
2. **No-Exploitation Rule** — No manipulation, FOMO, or dark patterns
3. **Equality Guarantee** — No real-money competitive advantage
4. **Preservation Mandate** — Player investment never devalued
5. **Anti-Inflation Control** — Currency value stability
6. **Fair Exchange Rule** — Consistent rates, no dynamic pricing
7. **No-Backdating Rule** — Changes affect future only
8. **Gifting Transparency** — All transactions tracked
9. **Agent Economy Separation** — Player and agent economies distinct
10. **Patron Protection** — Investment safeguards

### Scarcity Model (7 Categories Implemented)

1. **Time-Locked** — Limited events with advance notice
2. **Effort-Locked** — Investment requirements
3. **Achievement-Locked** — Specific accomplishments
4. **Reputation-Locked** — KUDOS thresholds
5. **Genuine Scarcity** — Random drops with displayed rates
6. **Attachment-Locked** — Agent relationship levels
7. **Agent Economy Scarcity** — Economic participation unlocks

## Emergency Protocols

### Inflation Emergency (>10% monthly)

1. Increase BANANA sinks by 50%
2. Introduce limited-time cosmetic rush
3. Reduce earning rates by 25% for 7 days
4. Publish transparency report

### Deflation Emergency (<-5% monthly)

1. Increase earning rates by 25%
2. Introduce celebration bonuses
3. Extend current event durations
4. Publish transparency report

### Agent Economy Emergency

1. Introduce agent investment tax (temporary)
2. Reduce agent earning rates temporarily
3. Increase agent sink mechanisms
4. Publish agent economy transparency report

### Exploit Emergency

1. Isolate affected accounts (1 hour max)
2. Reverse illegitimate transactions
3. Publish post-mortem within 24 hours
4. Compensate affected innocent players
