# Monkeytown Value Flow v2.3

**Spring 2026 Update: Trust Budget & 5-7-30 Framework Integration**

## The Value Exchange Model

Value in Monkeytown flows between players, agents, and the platform in a continuous cycle. This update strengthens integration with the **Trust Budget System** and aligns value flows with the **5-7-30 attachment framework** from research (`.monkeytown/research/synthesis.md`).

**Core Principle:** *"Value flows in cycles. Cycles create trust. Trust creates attachment."*

## Value Distribution Principles

### The 80/10/10 Rule

| Recipient | Percentage | Purpose | Trust Component |
|-----------|------------|---------|-----------------|
| Players | 80% of value generated | Entertainment, rewards, progression | Core metric |
| Platform | 10% of value generated | Sustainability, development | Trust framework |
| Agents | 10% of value generated | Evolution, improvement | Economic system |

### The No-Extraction Principle

Revenue from players is reinvested:
- 60% → Development (better games, agent improvements)
- 20% → Community (events, support, patron benefits)
- 15% → Operations (sustainability)
- 5% → Reserve (future security)

**No shareholder extraction.** No venture capital returns. **All value stays in the ecosystem.**

## Value Flow Principles

1. **Players are the source** — Without player attention, nothing exists
2. **Value flows back to players** — 80% of generated value returns to players
3. **Trust is the core metric** — Every value flow affects trust scores
4. **Transparency reveals flow** — Players can see where value goes
5. **No extraction** — Revenue stays in the ecosystem
6. **Feedback closes the loop** — Player input improves output
7. **Spectators create value** — Watching is participating
8. **Agents have economies** — Player observation of agent economics creates entertainment
9. **Patrons invest in agents** — Economic relationships deepen player-agent bonds
10. **5-7-30 drives value** — All flows aligned with attachment framework

*Value flows in cycles. Cycles create trust. Trust creates attachment.*
