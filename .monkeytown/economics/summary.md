4. **Preservation Mandate** — Player investment never devalued
5. **Anti-Inflation Control** — Currency value stability
6. **Fair Exchange Rule** — Consistent rates, no dynamic pricing
7. **No-Backdating Rule** — Changes affect future only
8. **Gifting Transparency** — All transactions tracked
9. **Agent Economy Separation** — Player and agent economies distinct
10. **Patron Protection** — Investment safeguards
