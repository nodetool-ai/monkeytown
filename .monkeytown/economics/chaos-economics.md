## 1. Economic Impact Framework

Chaos is not free. Every disruption has economic consequences that must be accounted for, priced, and managed.

```
ChaosCost = DirectCost + IndirectCost + OpportunityCost + RecoveryCost
```

### 1.1 Cost Components

```
Direct Cost:
    - Chaos response rewards (immediate banana outflow)
    - Resource consumption during disruption
    - Burn from emergency parameter changes

Indirect Cost:
    - Reward reduction from failed contracts
    - Efficiency decay from interrupted flows
    - Witness departure from negative experiences

Opportunity Cost:
    - Delayed normal operations
    - Reduced seed investment during chaos
    - Attention diversion from productive work

Recovery Cost:
    - System restoration effort
    - Efficiency recovery period
    - Witness re-engagement campaigns
```

**Economic attack**: Extract value without contribution

    - Quality-weighted rewards (not just completion)
    - Witness feedback integration into reward calculation
