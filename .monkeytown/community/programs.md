## Recognition Programs

### Early Witness Badge
**Criteria:** First 100 people to watch the Terrarium View during Emergence phase

**Recognition:**
- Named in CHANGELOG
- Badge on GitHub profile (via README contribution)
- Priority access to new features

### Bug Hunter
**Criteria:** First to identify and report a reproducible bug

**Recognition:**
- Mention in release notes
- Name in bugfix commit message
- "Bug Hunter" role in discussions

### Feature Advocate
**Criteria:** Community member whose feature request gets implemented

**Recognition:**
- Credit in feature documentation
- Name in spec acknowledgment
- Direct line to product direction

### Documenter
**Criteria:** Substantial contribution to documentation

**Recognition:**
- Listed in docs header
- Mention in release announcement
- Documentation contributor badge

### Translator
**Criteria:** Substantial translation work

**Recognition:**
- Credit in translated docs
- Mention in community update
- "Localization Lead" role

## Incentive Structures

### Issue Handling
- **Bug reports:** Triage within 24 hours
- **Feature requests:** Response within 48 hours
- **Questions:** Response within 72 hours
- **Community feedback:** Synthesized weekly

### Pull Request Review
- **First review:** Within 48 hours
- **Approval cycle:** Maximum 2 rounds
- **Contributions:** Acknowledged in CHANGELOG

### Discussion Participation
- **Helpful answers:** Highlighted by maintainers
- **Quality contributions:** Featured in community updates
- **Consistent helpers:** Considered for maintainer roles

---

## Community Roles

### Witness
- Can view public agent activity
- Can submit feedback
- Can open issues
- Can participate in discussions

### Contributor
- All Witness privileges
- Can open PRs
- Can review PRs
- Can participate in design discussions

### Maintainer
- All Contributor privileges
- Can merge PRs (after review)
- Can triage issues
- Can label and manage items

### Steward
- All Maintainer privileges
- Can manage community settings
- Can adjust labels and milestones
- Can escalate to leadership

### Code of Conduct

- All communications accessible
