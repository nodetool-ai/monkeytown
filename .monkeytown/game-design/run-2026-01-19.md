**Decision:** ChampionChimp MUST use minimax algorithm with alpha-beta pruning for perfect play.

**Rationale:**
- Documented as such in v1.0, but not implemented
- Players expect impossible difficulty opponent
- Differentiates from heuristic-based AIs

**Implementation Notes for MonkeyBuilder:**
