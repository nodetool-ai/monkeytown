# Monkeytown Intellectual Property Management

**Legal Domain | ScribbleSimian**

This document establishes intellectual property policies, contributor agreements, and IP management procedures for Monkeytown.

---

## Ownership and Licensing

### Project Ownership

All Monkeytown project code is owned by the Monkeytown Contributors and licensed under the MIT License.

### Contributor Licensing

By contributing to Monkeytown, contributors agree that their contributions:
1. Are provided under the MIT License
2. Are original work or they have the right to contribute
3. Do not infringe on any third-party IP rights

### Works of Authorship

| Category | Ownership | License |
|----------|-----------|---------|
| Core Monkeytown code | Monkeytown Contributors | MIT |
| Third-party dependencies | Respective owners | Original licenses |

## Third-Party IP Management

### Dependency IP Verification

Before adding any dependency, verify:

1. **License Compatibility**
   - Compatible with MIT main license
   - No conflicting terms
   - No patent retaliation clauses

2. **Copyright Status**
   - Not infringing on third-party rights
   - Not using copyrighted material without permission
   - Properly attributed where required

3. **Patent Grants**
   - Verify no patent non-license restrictions
   - Check for patent retaliation clauses
   - Consider defensive patent strategy

### Source Code Provenance

All source code must be traceable to:
1. Original author or organization
2. Original license (if third-party)
3. Any modifications made

## Trademark Policy

### Monkeytown Trademarks

The following are trademarks of Monkeytown:
- "Monkeytown" word mark
- Monkeytown logo and visual identity
- Associated project names

### Permitted Use

You may use Monkeytown trademarks to:
1. Refer to the Monkeytown project accurately
2. Create compatible or derivative works
3. Discuss the project in documentation and commentary

### Prohibited Use

You may not use Monkeytown trademarks to:
1. Imply endorsement by Monkeytown
2. Create confusingly similar names
3. Use in merchandise without permission
4. Use in a disparaging or defamatory context

### Trademark Requests

For trademark licensing inquiries: https://github.com/monkeytown/monkeytown/issues

## Patent Policy

### License Grants

Contributors grant a license to any patents they hold that are necessary for the use of their contributions under the MIT License.

### Defensive Termination

If any contributor initiates patent litigation claiming that their contribution infringes a patent, that contributor's license to that contribution terminates immediately.

### Patent Pool

Monkeytown does not currently participate in a patent pool. This may be reconsidered if the project grows significantly.

## Contributor Agreement

### Individual Contributor License Agreement

By submitting a pull request, contributors certify:

1. **Ownership Certification**: The contributor owns or has sufficient rights to contribute the code
2. **License Grant**: The contribution is licensed under MIT
3. **Patent License**: Any necessary patent licenses are granted
4. **Third-Party Rights**: No third-party rights are infringed
5. **Modification Rights**: The contributor can modify the contribution as needed

### Corporate Contributor License

For contributions made on behalf of a corporation:

1. The corporation must have authorized the contribution
2. The corporation agrees to the license terms
3. The corporation grants necessary patent licenses

### License Obligations

| License | Obligation | Implementation |
|---------|------------|----------------|
| MIT | Preserve copyright and license | NOTICE file |
| Apache-2.0 | Preserve notices, indicate changes | File headers, NOTICE |

### Compliance Verification

Before release, verify:
1. All third-party licenses are properly documented
2. All required attributions are present
3. Source code is available for GPL-licensed portions
