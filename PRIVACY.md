# Monkeytown Privacy Policy

**Last Updated: January 17, 2026**

## Overview

Monkeytown is an autonomous software experiment operated by the Monkeytown community. This policy describes how data is handled in connection with the project.

## Data We Collect

### Public Repository Data
All code, documentation, and discussions in the Monkeytown repository are public. This includes:
- Source code and commits
- Issue discussions and pull request conversations
- Agent output logs and decision records

### Automatically Collected Data
We may collect:
- Basic usage analytics (page views, repository clones)
- Aggregate contributor statistics
- Error and performance logs

### Data We Do Not Collect
- Personal identifying information (unless voluntarily provided)
- Private communications
- User credentials or authentication tokens

## How We Use Data

- To operate and improve the Monkeytown experiment
- To maintain code quality and project documentation
- To analyze project health and community engagement
- To respond to security concerns and incidents

## Data Sharing

We do not sell or trade personal data. Public repository data is available to anyone. We may share aggregate statistics in project reports.

## Data Retention

Public repository data is retained indefinitely. Automatically collected analytics data may be purged after 12 months.

## Third-Party Services

Monkeytown uses GitHub for hosting. GitHub's privacy policy applies to their services. See: https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement

## Children's Privacy

Monkeytown is not directed at children under 13. We do not knowingly collect information from children.

## Changes to This Policy

We may update this policy as the project evolves. Material changes will be announced in the repository.

## Contact

Privacy questions: https://github.com/monkeytown/monkeytown/issues
