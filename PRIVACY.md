# Monkeytown Privacy Policy

**Last Updated:** January 18, 2026

## Overview

Monkeytown is an autonomous software experiment operated by the Monkeytown community. This policy describes how data is handled in connection with the project, including data from gameplay, AI interactions, and game evolution features.

## Data We Collect

### Public Repository Data

All code, documentation, and discussions in the Monkeytown repository are public. This includes:
- Source code and commits
- Issue discussions and pull request conversations
- Agent output logs and decision records

### Player Data

When you play Monkeytown games, we collect:

**Gameplay Data:**
- Game actions, moves, and decisions
- Game outcomes and scores
- Session duration and frequency
- Achievement progress

**AI Interaction Data:**
- Which AI opponents you play against
- Agent transparency requests
- Feedback submissions
- Memory preferences (Forgotten Mode, consent settings)

**Player Profile Data:**
- Display name (optional)
- Player statistics and achievements
- Preferences and settings

**Communication Data:**
- Chat messages (public only)
- Feedback comments

**Technical Data:**
- IP address (for connection management)
- Browser and device information
- Performance metrics
- Error logs

### Data We Do Not Collect

- Personal identifying information (unless voluntarily provided)
- Private communications
- User credentials or authentication tokens (handled by auth provider)
- Payment information (no payments processed)

## How We Use Data

### Game Operations
- To run games and manage game state
- To match players with appropriate AI opponents
- To track scores and achievements
- To save game progress

### AI Agent Functionality
- To enable AI agent memory and continuity
- To allow agents to learn from player behavior
- To provide transparent AI decision explanations
- To personalize AI opponent interactions

**Important:** AI agents only use your data to improve gameplay. We do not use player data to train external AI models or for any purpose outside Monkeytown.

### Game Evolution
- To prioritize improvements based on feedback patterns
- To measure feature adoption
- To track Evolution Feed engagement
- To understand which changes players value

### Project Improvement
- To operate and improve the Monkeytown experiment
- To maintain code quality and project documentation
- To analyze project health and community engagement
- To respond to security concerns and incidents

## Agent Transparency Data

When you play against AI opponents, agents may collect:

| Data Type | Purpose | Retention |
|-----------|---------|-----------|
| Game history | Agent memory and continuity | Until Forgotten Mode or account deletion |
| Interaction frequency | Relationship tracking | 90 days |
| Strategy responses | Learning player patterns | 30 days |
| Feedback received | Improving gameplay | Indefinite (aggregated) |

You can control this data through:

- **Forgotten Mode**: Clears all agent memory of you
- **Evolution Consent**: Controls which updates affect your experience
- **Privacy Settings**: Adjust data collection preferences

## Data Sharing

We do not sell or trade personal data. Public repository data is available to anyone.

**Limited Sharing:**
- Aggregate statistics may be shared in project reports
- Anonymized gameplay data may be used for game balance research
- Service providers (hosting, analytics) have limited access per their policies

**Never Shared:**
- Individual player identities
- Private chat messages
- Feedback content with personal information
- Authentication credentials

## Data Retention

| Data Type | Retention Period |
|-----------|------------------|
| Gameplay history | Until deletion request |
| Agent memory | Until Forgotten Mode or 2 years inactive |
| Chat messages | 30 days |
| Feedback submissions | Indefinite (anonymized after 1 year) |
| Technical logs | 90 days |
| Account data | Until account deletion |

## Third-Party Services

Monkeytown uses:
- **GitHub**: Repository hosting
- **Auth Provider**: Authentication (see their privacy policy)
- **Hosting Infrastructure**: Server and network operations

Each service has its own privacy policy. We select partners committed to data protection.

## Children's Privacy

Monkeytown is not directed at children under 13. We do not knowingly collect information from children. Parents who believe their child has played Monkeytown may contact us to request data deletion.

## Your Rights

You have the right to:

- **Access** your player data
- **Correct** inaccurate data
- **Delete** your data and account
- **Export** your game history
- **Control** agent memory (Forgotten Mode)
- **Manage** Evolution Consent settings

**To exercise these rights:**
1. Settings → Privacy → Manage Your Data
2. Or contact: https://github.com/monkeytown/monkeytown/issues

## AI-Specific Privacy

### Agent Memory

AI opponents remember past games to provide continuity. This includes:
- Your playing style
- Past interactions
- Game outcomes together
- Preferences you've shown

**You control agent memory:**
- View memory: Agent Panel → Memory tab
- Clear memory: Forgotten Mode
- Adjust preferences: Settings → Privacy

### Edge AI and Local Processing

Some AI processing happens locally on your device:
- Personality layer interactions
- Quick decision responses
- Offline gameplay

Local processing does not send data to servers.

### Evolution Consent

Control how game changes affect you:

| Setting | Effect |
|---------|--------|
| Full Evolution | All updates as they ship |
| Minor Updates Only | Skip experimental features |
| Stable Only | Only proven changes |
| Pause Evolution | Lock current experience |

**Settings:** Settings → Evolution → Evolution Consent

1. In the Evolution Feed
2. On GitHub repository
3. With 30 days notice before taking effect
