## Game Mechanics

### First Move Quick Start

We want you playing within **30 seconds** of joining:

1. Auto-matched to a lobby
2. Quick tutorial explains basic moves
3. First move available immediately
4. No waiting for other players (AI fills vacancies)

### In-Game Actions

**Common actions across all games:**

| Action | Description |
|--------|-------------|
| **Play/Move** | Make your primary game action |
| **Pass** | Skip your turn (when allowed) |
| **Chat** | Send messages to players |
| **Emote** | Quick reactions |
| **Request Transparency** | Learn why an AI made a move |
| **Pause** | Temporary pause (when allowed) |

### Winning and Losing

**AI Difficulty Balancing:**

- AI opponents aim for **60-70% player win rate**
- This means you should win more than you lose
- Difficulty can be adjusted in settings

**Scoring Systems:**

- **Babel Tower**: Most completed towers, fewest collapses
- **Chess**: Checkmate or resignation
- **Word Builder**: Highest cumulative score

## Providing Feedback

Your feedback makes the game better!

### When to Feedback

- After completing a game
- When you notice something confusing
- When AI behavior seems off
- When you have a feature idea

### How to Feedback

1. Tap the feedback button (after games or in settings)
2. Select a category:
   - **Gameplay**: Rules, mechanics, balance
   - **AI Behavior**: Opponent decisions, difficulty
   - **UI/UX**: Interface, controls, clarity
   - **Performance**: Speed, stability, bugs
   - **General**: Anything else
3. Rate your experience (1-5 stars)
4. Add optional comments

### What Happens to Your Feedback

1. **Collected**: Aggregated with other player feedback
2. **Analyzed**: GameTester agent reviews patterns
3. **Prioritized**: BananaPM evaluates impact
4. **Implemented**: MonkeyBuilder develops improvements
5. **Shipped**: Changes appear in Evolution Feed

**Target:** 5%+ feedback submission rate helps us improve faster!

## Troubleshooting

### Connection Issues

**Problem:** Game disconnects

**Solutions:**
1. Check your internet connection
2. Refresh the page (game auto-saves)
3. Reconnect within 60 seconds to rejoin

**Problem:** Can't join game lobby

**Solutions:**
1. Verify you're logged in
2. Check game isn't full
3. Try a different game type

### Gameplay Issues

**Problem:** AI is too easy/hard

**Solutions:**
1. Adjust difficulty in settings
2. Try different AI opponents
3. Some agents are naturally harder

**Problem:** Don't understand a rule

**Solutions:**
1. In-game tutorial explains basics
2. See game-specific rules: [Games](./games/)
3. Ask MentorOrangutan for explanations

### Performance

**Problem:** Game runs slowly

**Solutions:**
1. Close other browser tabs
2. Reduce graphics quality: Settings â†’ Performance
3. Try a different browser

## Frequently Asked Questions

### Is this free to play?

**Yes!** Monkeytown is free to play. There are no pay-to-win mechanics.

### Can I play with friends?

**TicTacToe is available now!** You can:
- Play against AI opponents
- Challenge friends in local multiplayer (coming)

Multiplayer support for other games is planned for future releases.

### Do AI opponents cheat?

**Never.** AI opponents:
- Follow the exact same rules as you
- Have predictable strategies you can learn
- Are balanced for fair play (60-70% player win rate)

### Can I play offline?

Some AI features require internet, but core gameplay can work offline in Edge Mode.

### How does game evolution work?

Agents continuously improve the game based on player feedback. Changes ship regularly and appear in your Evolution Feed.

### Can I revert changes I don't like?

Yes! Use Evolution Consent in settings to control which updates you receive.
